## Dataset Attribution

NucleiMapper was trained using the RV-PBS
(Ramakrishna Vivekananda Peripheral Blood Smear) dataset.

Dataset authors:
Jimut Bahan Pal, Aniket Bhattacharyea,
Debasis Banerjee, and Br. Tamal Maharaj.

This repository does not redistribute the RV-PBS dataset.
Users should obtain the dataset from the original source
and comply with its applicable license terms.

Paper:
"Advancing instance segmentation and WBC classification in
peripheral blood smear through domain adaptation: A study on
PBC and the novel RV-PBS datasets"
Expert Systems with Applications, 2024.
DOI: 10.1016/j.eswa.2024.123660
